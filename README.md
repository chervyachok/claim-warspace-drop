Инструкция .......................

Установить node js

далее в папке с ботом открыть powershell или cmd

npm install


Конфигурация .....................

В файле claim.js в privatekey вставить свой приватный ключ
В data в ''actor'' вставить имя своего аккаунта , в asset_ids в квадратные скобки поставить id нфт от варспейс которое вы храните
Saletime настроен , для теста можете указать другое время
howearly наскольно раньше толкнуть транзакцию в сеть(миллисекунды), рекомендую ставить хотя бы 800-900

Запуск ........................


Открыть в папке с ботом powershell или cmd

За минуту до дропа

npm run claim


Важно: нужно сделать депозит на нефти заранее + бот будет пытаться брать сразу 4 пака
